Debian Jessie 切换 systemd

June 13, 2014 @ 12:56 PM

目前，许多 Linux 发行版都相继切换到了 systemd。在 Debian Jessie
中，要从 SysVinit 切换到 systemd，可执行如下命令：

apt-get update
apt-get install systemd systemd-sysv
reboot

在切换前，注意查看系统是否满足需求。
